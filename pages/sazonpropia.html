Sazon Propia

Diseño de página de Recetas

Febrero 2023

Sazón propia, es un proyecto académico basado en la creación de una pagina Web de Recetas utilizando HMTL y CSS. En este proyecto participé como co-creadora de la idea. Realizamos un maquetado de la página apoyándonos de la herramienta Balsamiq para realizar bosquejos de los distintos Wireframes. Posteriormente realizamos los documentos HTML y aplicación de estilos con CSS.

Diseñando la Identidad: Colores

Para este proyecto quisimos elegir imagenes vivas y un color principal que resultara llamativo y estuviera relacionado con el área gastronómica. En este sentido, el color verde resultó ideal ya que permite resaltar la imagen seleccionada para el home y hace buen contraste con el resto de la galería.

Diseñando la Identidad: Logo

Después de haber seleccionado este color principal para la Web, diseñamos un logo que evocara el concepto principal de la página que es inspirar a realizar preparaciones propias inspirándose en recetas replicadas pero personalizadas. Por eso la elección de la frase Sazón Propia.

La sección "Nosotros" fue diseñada para dar una breve descripción del origen y finalidad de la página. Incorpora imágenes referenciales de entornos familiares ya que el concepto de la Web es ofrecer recetas de sencilla preparación para los menos experimentados.

La seccion "Contáctanos" incorpora un formulario que permite al visitante comunicarse con los administradores de la Web. Para ello requiere ingresar algunos datos personales y dispone además de un espacio para escribir comentarios o solicitudes.

Para la sección de Recetas, quisimos que resultara llamativa e intuitiva. Ideamos un menú lateral que permita acceder desde una misma vista a todas las recetas, de este modo facilitaría la navegación. Se utilizaron imágenes coloridas para hacer referencia a sencillas y sabrosas preparaciones. Dispone además de una opción descargable de las recetas para quienes deseen disponer del documento en su equipo o para imprimir.
