Sazón Propia

Diseño de página de Recetas

Febrero 2023

Sazón propia, es un proyecto académico basado en la creación de una página Web de Recetas utilizando HMTL y CSS. En este proyecto participé como co-diseñadora. Realizamos un maquetado de la página apoyándonos de la herramienta Balsamiq para construir los distintos Wireframes. Posteriormente realizamos los documentos de código HTML y aplicación de estilos con CSS.

Diseñando la Identidad: Colores

Para este proyecto elegimos imágenes vivas y un color principal que resultara llamativo, enérgico y relacionado a comida sana. En este sentido, el color Amarillo Verdoso resultó ideal ya que se vincula con alimentos saludables, inspira energía y además resalta la galería de imágenes seleccionadas para esta Web.

Diseñando la Identidad: Logo

Después de haber seleccionado el color principal para la Web, diseñamos un logo que evocara el concepto principal de la página que es inspirar a realizar preparaciones propias basándose en recetas replicadas pero que puedan ser personalizadas. Por eso la elección de la frase Sazón Propia.

La sección "Nosotros" fue diseñada para dar una breve descripción del origen y finalidad de la página. Incorpora imágenes referenciales de entornos familiares ya que el concepto de la Web es ofrecer recetas de sencilla preparación y para todo el mundo.

La sección "Contáctanos" incluye un formulario que permite al visitante comunicarse con los administradores de la Web. Para ello requiere ingresar algunos datos personales y dispone además de un espacio para escribir comentarios o solicitudes.

La sección de Recetas, quisimos que resultara llamativa e intuitiva. Por esto creamos una página denominada Galería, donde se disponen cartas contenedoras de imágenes relativas a las recetas disponibles y un botón que te dirije a la misma. Las propias recetas se presentan en una tarjeta amplia, que contiene un menú lateral que permite acceder desde una misma vista a todas las recetas, de este modo facilita la navegación. Se utilizaron imágenes coloridas para hacer referencia a sencillas y sabrosas preparaciones. Dispone además de una opción descargable de las recetas para quienes deseen disponer del documento en su equipo o para imprimir.
